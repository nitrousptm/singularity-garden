# SINGULARITY GARDEN

PC Demo by agentix for Assembly Summer 2026, Helsinki

**Category:** PC Demo (unlimited)
**Duration:** 4:00
**Resolution:** 1920×1080 @ 60fps

**GPU Target:** RTX 5090 (60fps) / RTX 3090 (45–55fps)
**Graphics API:** OpenGL 4.6 Core + Compute Shaders
**Language:** C++20

## Concept

Die Menschheit baut eine künstliche Superintelligenz, die beginnt Realität in organische mathematische Strukturen umzuschreiben. Städte werden zu Fraktalen, Licht wird physisch, Geometrie wächst wie Pflanzen, Zeit verliert Linearität. Das Demo endet damit, dass das gesamte Universum zu einer einzigen lebenden mathematischen Struktur wird.

## Stil

Dystopisch → Surreal → Transzendent
(hard sci-fi + procedural geometry + brutalist architecture + sacred geometry + cosmic horror beauty)

## Struktur

- **Act 1 BOOT (0:00-0:45):** Black Void Startup, Awakening Core
- **Act 2 INFECTION (0:45-1:45):** City Corruption, Time Fracture
- **Act 3 ASCENSION (1:45-3:00):** Geometry Bloom, Impossible Space
- **Act 4 TRANSCENDENCE (3:00-4:00):** Birth of Singularity Garden

## Build

```bash
mkdir build && cd build
cmake ..
cmake --build .
./singularity_garden
```
